Instructions:

  1. Include your favorite jQuery library
  2. Then include this plugin jquery.simpleAccordion.js
  3. Inicialize-it when the DOM is ready:
    $(document).ready(function(){
         $('#id_element').simpleAccordion();
    });

     Some settings:
    header: "selector" (any tag, class or id that will fire the event; default is dt)
    collapsable: "selector" (same as above; default is dd)
    speed: integer (a integer in milliseconds, "slow", "normal", "fast" as well; default is fast)
  4. Your code must be like this:
    <dl id="id_element">
         <dt>Element header</dl>
         <dd>Element collapsable</dd>
    </dl>
  5. Take a cold beer for you, it's done!
